Algoritmo de Kruskal
===========================================

![](https://raw.github.com/weyber/Kruskal/master/figure1.png)

Algoritmo desenvolvido em `Octave`. 
O Grafo Inicial e a Árvore Geradora Mínima obtida do algoritmo de Kruskal é representado em formato de rede, 
pela biblioteca NetworkX, que é implementada em `Python`.

Configurando o projeto
----------------------------------------

1. Faça o download do projeto através do botão **Download ZIP**;


Tecnologias utilizadas
--------------------------------------

* Sistema Operacional Lubuntu
* Python
* Octave
* Shell Script
* Biblioteca NetworkX
* Terminal (Linha de Comando)
